# 🏨 Hostel Finder

A web application that helps users find hostels or rooms near them when visiting a new place. Users can view hostel details, read reviews from previous residents, and contact hostel owners directly.

---

## 🚀 Features

* 🔍 Search hostels near your location
* 🏨 View hostel details and facilities
* ⭐ Reviews from previous hostel members
* 📞 Contact hostel owner (phone/email)
* ❤️ Save favorite hostels
* ➕ Add new hostel listing
* 🧑‍💼 Admin / owner dashboard
* 📱 Responsive design

---

## 🛠️ Tech Stack

* JavaScript
* JSON (Data Storage)

## ▶️ How to Run Locally

1. Clone repository

```
git clone https://github.com/YOUR_USERNAME/Hostel-finder.git
```

2. Open folder

```
cd Hostel-finder
```

3. Install dependencies

```
npm install
```

4. Run project

```
npm start
```
